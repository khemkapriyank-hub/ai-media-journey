What is a DataFrame? A DataFrame is a table-like data structure in Python (similar to Excel) that allows easy filtering, analysis, and manipulation of data.
How is filtering different from loops? A loop processes data row-by-row manually, while filtering applies a condition to the entire dataset at once and returns a subset of matching rows.
What does df["Fill_Level"] < threshold do? checks each row in the Fill_Level column and returns True/False values indicating which rows meet the condition.
